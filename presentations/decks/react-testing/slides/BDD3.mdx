# Benefits of BDD

<ul>

<Appear>

  <li>Allows better communication between customers/developers/testers</li>
  <li>The step library gradually becomes bigger which makes it easier to write new test cases</li>
  <li>Customers etc. can be enabled to write tests without actually knowing how to program</li>
  <li>This in turn can lead to a better test coverage and a shared understanding of why tests are so useful</li>

</Appear>

</ul>
