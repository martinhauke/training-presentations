# React-testing-library

<ul style="font-size: 40px;">

<Appear>

  <li>library to create component/integration/(e2e) tests</li>
  <li>uses the DOM Api provided by Jest by default</li>
  <li>goal is to test from the users perspective</li>
  <li>prior libraries like <strong>enzyme</strong> where more jquery like, fetching dom nodes by implementation details</li>
  <li>Three query types: <i>get</i>, <i>find</i>, <i>query</i></li>
  <li><i>get</i> is synchronous and throws an error if the element is not found</li>
  <li><i>find</i> is asynchronous (returns a promise) and also throws</li>
  <li><i>query</i> is synchronouse and returns <i>null</i> element is not found</li>
  <li><i>find/getbyRole</i> as sane default</li>
  <li><i>query</i> only for testing if something does not exist</li>

</Appear>

</ul>
