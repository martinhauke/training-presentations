## Reselect

<ul>

<Appear>

<li>Memoize Selectors</li>
<li>Automatically included in Redux-Toolkit</li>
<li>
    <ul>
        <li>Passing all parameters to all "input selectors"</li>
        <li>Saving all the input selector results to an array</li>
        <li>Checking to see if any input results changed by reference</li>
        <li>If so, passing all input results to the output selector</li>
        <li>If not, return the previous result of output selector</li>
    </ul>
</li>

</Appear>
