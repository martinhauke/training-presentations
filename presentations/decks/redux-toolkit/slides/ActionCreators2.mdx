# createAction()

<ul>
<Appear>

<li>Combines the definition of the <i>actionCreator</i> with the definition of the type constant</li>
<li>The action type can then be referenced by calling either <i>someActionCreator.type</i> or <i>someActionCreator.toString()</i></li>
<li><strong>Note:</strong> <i>someActionCreator.type</i> only works with <i>createReducer()</i> / <i>createSlice()</i> not with regular reducers</li>
<li>Optionally a <i>prepare()</i>-callback can be used as second argument, to define how the payload should be processed</li>

</Appear>

</ul>
