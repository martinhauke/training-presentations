# Other useful API functions

- _createEntityAdapter()_ gives easy access to CRUD-operations and selectors for entity objectsa (e.g. Users, Posts, etc.), where each entity has to have an **id** property
- _createAsyncThunk()_ for creating more complex asynchronous actions
- _createAsyncThunk()_ creates multiple promise lifecycle action types (`pending`, `fulfilled`, `rejected`) which can then be handled inside a reducer
