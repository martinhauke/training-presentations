### mapStateToProps

<ul>

<Appear>

<li><code>mapStateToProps(state, ownProps?) => Object</code></li>
<li>Select parts of the data from entire state object</li>
<li>returns an Object</li>
    <li>Each field in the object will become a Prop for the wrapped Component</li>
    <li>Remember: The values in the fields (Props) will be used to determine if your component needs to re-render</li>

</Appear>

</ul>
